## Progress summary

### 0. Environment setup
- Cloned this repo; installed Git (winget) on the workstation.

  connect through `ssh-agent`.

### 1. Products fly-out (mega menu) — fixed on all 6 locales
- Root cause: the English "Products" menu item (104) had Max Mega Menu `type=megamenu`
  instead of `type=grid` (FR/ES/PL used `grid`), so the fly-out never rendered on EN
  (and the EN-alias locales en-ca/en-uk).
- Fix: set item 104 → `type=grid` (referencing the `[wj_megamenu]` block widget). Verified
  the panel now renders identically on en-us, es-us, fr-ca, pl-pl, en-ca, en-uk.

### 2. Header — full port + nav restructure (all locales)
- Replaced the old single-row header with the blueprint's **two-row Figma header** (blue
  utility bar + white main nav + locale-aware CTA), ported `header.css` (1548 ln) + mobile
  JS. Adapted: base menu locations (kept `primary`/`header-nav` + plugin remap so MMM keeps
  working), preserved dev GTM, dropped axyz's Zapier chatbot, `routers`→`products` selectors.
- **English nav reordered** to axyz structure: main = Products, Industries, Features,
  Software, Support, About Us; upper = **Parts→"Buy Parts"** (cncshop.com), **Resources**
  (dropdown: News & Events, Blogs, Webinars, Brochures), Locations, [lang], [search].
  Removed "Our Sites" + "Learn" (merged into Resources); removed the "Buy Parts" button
  from the header CTA (now only "Get a Quote"). Upper-nav hover matched to axyz.
- Set the **dark WARDJet logo** in the header (old white logo was invisible on the white bar).

### 3. Homepage — rebuilt section-by-section (new section-based template)
