## Progress summary (localization QA pass)

### 1. translation_group_id re-keyed to axyz format
- axyz uses `{post-slug}-{year}` (full slug, no truncation) + legacy `wpml-{trid}`.
  wardjet had raw numeric IDs.
- Re-keyed **554 groups / 1318 posts** to `{en-us-slug}-{year}` (year = en-us post
  year). 77 duplicate-slug blog groups disambiguated with `-{pid}`. Siblings stay
  linked; routing + language switcher + CTA resolution all verified. Backup:
  `tgid-backup.json`.

### 2. Localized homepage sections (video + features)
- **Video hero** (`video_carousel`) and **Features slider** (`features_slider_items`)
  were only on en-us → populated on es/fr/pl/en-ca/en-uk (video copied; features
  built by matching accessory image IDs to each locale's translated gallery).
- Fixed **5 English feature cards** on es/fr (en-only image IDs 6806–6810 absent from
  localized galleries) by mapping to the localized accessory entries; "Flip Up Guards"
  → localized Splash Guard. Only "SmartRemove" stays English (brand name).

### 3. Menu label accuracy (from blueprint)
- Extracted axyz's real menu labels from the dump (resolving post_type links to their
  linked page titles). Corrected header-nav: **Buy Parts** es "Compre Partes", pl
  "Zamów części"; **Resources** children es Webinars "Seminario web", fr "Webinaire",
  pl "Seminarium internetowe", en "News & Events".
- Added **Testimonials** subitem under Resources (all locales; localized link — locale
  archive where it exists, else /us/en/). Reordered Resources: News, Blogs, Brochures,
  Testimonials, Webinars.

### 4. Mega-menu (MMM) fixes
- The earlier menu rebuild dropped the Max Mega Menu `_megamenu` postmeta on es/fr/pl
  main-nav items → **desktop chevrons appeared** and the **Products grid panel** became
  a plain flyout. Copied EN's settings (`hide_arrow` + grid+widget on Products) → both
  fixed on all locales.

### 5. Footer-main group reorder
- Reordered all locales to: Products, Industries, Options, Software, Support, About Us,
  Companies.

### 6. Homepage content localized (per locale, from blueprint or translated)
- **Icon-strip** labels/subtitles (Líder de la Industria / Leader de l'Industrie /
  Lider Branży, …).
- **Products section** title/subtitle: template now reads `product_heading`/`product_copy`
  (was hardcoded English fallback); values set. **Group titles** (Abrasive/Water Only)
  localized via i18n map from category names; renamed the pl abrasive category term 78
  "Bez kategorii" → "Systemy ścierne" (fixes the products page too).
- **Learn More** card labels (Aprende más / En savoir plus / Dowiedz się więcej).
- **Partnerships** subtitle (translated — no blueprint source).
- **Contact section** heading + subtitle (Contáctenos / Contactez-nous / Skontaktuj się
  z nami + copy) — set at page level (page override beats the gettext-translated fallback).

### 7. Accessories gallery scroll-to-feature (committed)
- Feature-card links (`#feature-{slug}`) had no target — added
  `id="feature-{sanitize_title(title)}"` + `scroll-margin-top` to each gallery card, so
  clicking a feature scrolls to its row on every locale.
